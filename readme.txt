== Frequently Asked Questions ==

=What is Timeline for WP Elementor ?=
Timeline for WP Elementor  is a WordPress plugin that allows you to create responsive and customizable timelines on your website. It is an add-on for the popular page builder plugin, Elementor, and provides you with a variety of options to customize your timelines.

=How does Timeline for WP Elementor  work?=
To use Timeline for WP Elementor , you need to install the plugin and activate it. Once you've done that, you can create a new timeline by adding a new Elementor section to your page and selecting the Timeline widget. From there, you can add and customize the timeline events, including text, images, icons, and links.

 =Can I use the plugin without Elementor Page Builder?=
No. You cannot use without Elementor since it's an addon for Elementor.

 =Does it work with any theme?=
Yes! It will work with any theme where Elementor works.
